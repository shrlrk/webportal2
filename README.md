# 학교생활+

학생과 교사만 이용할 수 있는 학교생활 전용 플랫폼입니다.

학교생활에 필요한 공지사항, 학생지원 서비스, 각종 신청 기능을 하나의 플랫폼으로 통합하여 최소한의 클릭만으로 원하는 기능에 접근할 수 있도록 설계하였습니다.

또한 최초인증 시스템을 적용하여 최소한의 개인정보만 활용하면서도 안전하게 서비스를 이용할 수 있도록 개발하였습니다.

---

## 주요 특징

- 최초인증 기반 사용자 인증
- 학생·교사 역할(Role) 기반 권한 관리
- 심플하고 직관적인 UI/UX
- 학교생활 관련 서비스를 하나의 플랫폼으로 통합
- 학년도별 진급을 고려한 사용자 관리 구조

---

## 주요 기능

### 로그인 및 사용자 관리

- 최초인증
- 학번(교번)·비밀번호 로그인
- 학년도별 진급 처리
- 학생·교사 권한 관리

### 학교생활

- 통합 게시판
- 즐겨찾기

### 학생지원

- 학생지원 서비스

### 신청 서비스

- 상담 신청
- 희망도서 신청
- 프로그램 신청
- 기타 학교 운영 신청

|------|------|

---

## 프로젝트 목표

학교에서 자주 사용하는 다양한 서비스를 하나의 플랫폼으로 통합하여 학생과 교사의 접근성을 높이고, 최소한의 개인정보만 활용하는 안전한 학교생활 서비스를 제공하는 것을 목표로 합니다.
